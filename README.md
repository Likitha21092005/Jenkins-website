# Simple Login Page

This is a basic login page built with HTML and CSS.

## Files

- `index.html`: The main HTML file containing the login form.
- `style.css`: Stylesheet for the login page.
- `README.md`: Project overview and instructions.

## How to Use

1. Clone or download the project.
2. Open `index.html` in your browser.
3. Customize the form action and backend logic as needed.

## Preview

A clean and responsive login form with username and password fields.
